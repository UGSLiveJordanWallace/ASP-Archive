Linux

With linux, downloading AStudyPal is primarily through the terminal, open the terminal by either using this shortcut

Ctrl+Alt+T

Or by navigating to your terminal


Then run this command in the terminal

$chmod +x linux-installer.sh

Followed by this command

$./linux-installer.sh

This converts the installer into an executable.


If the above throws an error just run this in the terminal

$sh linux-installer.sh


You will then be asked to enter your password in like this.

$Password:

After you have done this successfully, AStudyPal will be installed along with its dependencies. You should then see your desktop launcher on your desktop along with the executable in your user folder

Android

When downloading the android version of AStudyPal, there is a prerequisite you will need in order to successfully download this application.

This is the unknown sources feature that androids have in order to download apps like AStudyPal.


First You will have to navigate to your settings app on your phone.


Then navigate to the Biometrics and Security tab in your settings menu.


Then, navigate to the Install unknown apps tab

This brings up a list of apps that will need your permission to access this feature


What we choose from this tab depends on what browser or app you are using to download AStudyPal from. We will be choosing chrome for this tutorial.

BUT, the intructions that we will be showing you won't be to hard to follow when choosing a different app.


The section should look something like this


Turn on permissions


After the above has been down successfully, you can finally download AStudyPal.

Return to the website and hit the Android button above in the download section.

After this the download wizard will popup, follow the wizard and AStudyPal will installed
